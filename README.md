# TaskManager (EF Core + SQLite)
Мини-приложение на .NET 10 с использованием Entity Framework Core и SQLite.
Позволяет создавать задачи, хранить их в базе и работать с миграциями.

Приложение содержит простой REST API для задач:

Получить все задачи; Создать новую задачу; Обновить задачу; Удалить задачу

# Требования

* .NET 10 SDK

* SQLite (файл БД создаётся автоматически, отдельная установка не требуется)

* Git (для клонирования репозитория)

# Быстрый старт

* Клонируем репозиторий:

git clone https://github.com/yourusername/TaskManager.git

cd TaskManager
*********

* Восстанавливаем зависимости:

dotnet restore
*********

* Создаём базу и применяем миграции:

dotnet ef database update

После этого в корне проекта появится файл tasks.db.
*********
* Запускаем приложение:

dotnet run

Приложение будет слушать на http://localhost:xxxx

http://localhost:xxxx/swagger - откроет сваггер с апи
